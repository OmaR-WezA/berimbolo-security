لرفع التعديلات التي قمت بها على مشروعك Next.js إلى GitHub، يمكنك اتباع الخطوات التالية:

1. التحقق من حالة التعديلات
أولاً، تأكد من أنك داخل مجلد المشروع على جهازك:

bash
Copy code
cd path/to/your/project
ثم تحقق من حالة الملفات المعدلة باستخدام الأمر:

bash
Copy code
git status
سيعرض لك الملفات التي تم تعديلها أو إضافتها.

2. إضافة التعديلات إلى Git
إذا كانت لديك تعديلات، قم بإضافتها باستخدام:

bash
Copy code
git add .
أو إذا كنت ترغب في إضافة ملف معين فقط:

bash
Copy code
git add <اسم_الملف>
3. إنشاء Commit
بعد إضافة التعديلات، قم بإنشاء commit مع رسالة توضح التعديلات التي قمت بها:

bash
Copy code
git commit -m "Message describing your changes"
4. رفع التعديلات إلى GitHub
أخيرًا، قم برفع التعديلات إلى المستودع على GitHub باستخدام الأمر:

bash
Copy code
git push origin main
(إذا كنت تستخدم فرعًا آخر غير main، استبدل main باسم الفرع الذي تعمل عليه).

5. التحقق من التحديثات
بعد رفع التعديلات، يمكنك التحقق من GitHub لعرض التغييرات على المستودع. ستتمكن من رؤية التعديلات التي تم رفعها مباشرة.
